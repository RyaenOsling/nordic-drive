# Nordic Drive — Premium Car Rental Portfolio Project

**Nordic Drive** — это изысканный концептуальный веб-сайт для аренды премиальных электромобилей, выполненный в минималистичном скандинавском стиле (*Nordic & Clean*). 

Этот проект разработан для демонстрации навыков чистого проектирования интерфейсов (UI/UX) и качества фронтенд-кода: от структуры семантической разметки HTML5 и модульных CSS3 стилей по методологии БЭМ до реактивного управления состоянием интерфейса на чистом JavaScript (ES6+).

---

## 🎨 Дизайн и Визуальный Язык

- **Эстетика:** Максимум воздуха, тонкие линии, аккуратная типографика и приглушенные органические оттенки.
- **Цветовая палитра:**
  - Теплый молочный фон (`#F6F5F2`) вместо холодного чисто-белого.
  - Контрастный графитовый текст (`#18181A`) для безупречной читаемости.
  - Приглушенный хвойно-зеленый (`#2D4239`) и терракотовый (`#C56E52`) для интерактивных элементов и ошибок.
- **Типографика:**
  - *Cormorant Garamond* — элегантная контрастная антиква для заголовков, создающая ощущение премиальности и журнальной верстки.
  - *Plus Jakarta Sans* — современный геометрический гротеск для интерфейсных данных, форм и таблиц.

---

## 🛠️ Особенности реализации и Функционал

1. **Асимметричная Hero-секция:** Нестандартная сетка с крупным манифестом, художественным рендером Polestar 3 и лаконичной статистикой бренда.
2. **Интерактивный каталог с фильтрами:** Мгновенная фильтрация флота по классу (седан, внедорожник, туринг), типу трансмиссии и сортировка по цене без перезагрузки страницы.
3. **Бесшовное раскрытие деталей (CSS Grid Expand):** При клике на карточку автомобиля сетка каталога плавно раздвигается, и прямо внутри ряда встраивается подробный блок характеристик с интерактивной галереей (смена цвета вектора автомобиля при переключении ракурсов).
4. **Инлайн-визард бронирования (3 Шага):**
   - **Шаг 1 (Даты):** Выбор интервала аренды с мгновенной валидацией дат (дата окончания не может быть раньше начала).
   - **Шаг 2 (Инфо):** Минималистичная форма личных данных с интерактивной валидацией полей (имя, email, телефон).
   - **Шаг 3 (Оплата):** Динамический расчет стоимости аренды на основе тарифа автомобиля и количества выбранных дней, подробный чек и красивый экран успеха.

## 💻 Технический стек

- **HTML5:** 100% семантическая валидная разметка (теги `<header>`, `<main>`, `<section>`, `<nav>`, `<svg>`).
- **CSS3:** Без использования Tailwind / Bootstrap. Кастомные CSS-переменные для дизайн-токенов, адаптивная сетка CSS Grid (`repeat(auto-fill, ...)`), плавные аппаратные переходы (`cubic-bezier`). БЭМ-структура классов.
- **JavaScript (ES6+):** Чистый JS без библиотек (Vanilla JS). Собственное реактивное управление DOM и состоянием формы.
- **Тестирование (TDD):** Написан легковесный Node.js скрипт `test.js` с использованием встроенного модуля `assert`, покрывающий проверку файловой структуры, наличие переменных стилей, валидность HTML и корректность логики фильтрации и расчета стоимости.

## 🚀 Запуск и Тестирование

### Локальный запуск
Поскольку проект написан на чистом HTML/CSS/JS без сборщиков, достаточно открыть файл `index.html` в любом современном браузере.

### Запуск тестов
Для валидации корректности логики и структуры проекта выполните команду в консоли (требуется установленный Node.js):
```bash
node test.js
```
